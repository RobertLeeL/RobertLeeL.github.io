学习微信小程序开发

Posted on 2019-03-05

第一天 微信小程序框架

相关文件

主体文件

文件名 作用
app.js 主逻辑文件，用来注册小程序
app.json 主配置文件，用来对小程序进行全局配置
app.wxss 主样式表文件 设置样式

页面文件

文件名 作用
js 逻辑文件，编写js语言代码
wxml 页面描述文件，设置页面的布局，进行数据绑定
wxss 主样式表文件 设置样式
json 配置文件
配置文件

在小程序中需要使用到得页面都必须在app.json文件中进行配置。

配置状态
属性 作用
backgroundColor 设置窗口的背景色，用RGB表示
backgroundTextStyle 设置下拉刷新字体 loading图的样式
enablePullDownRefresh 是否开启下拉刷新
navigationBarBackgroundColor 设置导航栏背景颜色
navigationBarTextStyle 设置导航栏标题颜色
navigationBarTitleText 设置导航栏标题文字
tabBar 底部Tabbar 属性就不介绍了
connectSocket 网络请求超时时间
downloadFile 下载文件超时时间
uploadFile 上传文件超时时间
request 网络请求接口超时时间
逻辑层

生命周期函数：

函数 时机
onLaunch 当小程序初始化完成时调用方法
onShow 小程序启动，或者从后台到前台时调用方法
onHide 当页面隐藏时调用的函数
onReady 页面初次渲染完成的时候调用该函数 只会调用一次
onUnload 页面卸载的时候调用该函数
onLoad 页面加载完成的时候调用 只会调用一次
onPullDownRefresh 下拉刷新时调用的函数
